# Digital Friction Papers

This folder contains papers related to digital friction, with a focus on older adults.

## Structure

- `older_adults/`: papers centered on older adults and digital friction adjacent topics
- `general/`: broader digital friction papers for conceptual framing
- `download_manifest.tsv`: bilingual catalog used for download attempts
- `download_report.tsv`: final status report for each paper

## Status

- `downloaded_pdf`: a PDF was downloaded successfully
- `metadata_only`: the publisher or repository page was reachable, but no downloadable PDF was obtained in this run

Each item also has a `.txt` metadata file containing:

- English title
- Chinese title
- source page URL
- attempted PDF URL
- short notes
