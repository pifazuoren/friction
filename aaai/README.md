# AAAI 2026 LaTeX Samples

This folder stores AAAI-26 formatting samples for local reference.

## Official Overleaf Sample

Path:

- `official_overleaf_sample/`

Contents:

- `aaai-2026-press-formatting-instructions.pdf`
- `aaai-2026-overleaf-template-page.html`
- `anonymous-submission-latex-2026.tex`

Source:

- https://www.overleaf.com/latex/templates/aaai-2026-press-formatting-instructions-for-authors-using-latex/qnpmwrzmddjj

Note: the Overleaf page states that the source is the official AAAI author kit. The old short URL `https://aaai.org/authorkit26/` currently returns 404, so the public Overleaf template was saved as the stable accessible copy.

## Complete Local Compile Reference

Path:

- `unofficial_complete_latex_unified/`

Contents:

- `AAAI-2026-Latex-Unified-main.zip`
- extracted `AAAI-2026-Latex-Unified-main/`

This copy includes `aaai2026.sty`, `aaai2026.bst`, `aaai2026.bib`, and unified template files. Treat this as a convenience mirror for local compilation, not as the canonical AAAI source.

Before final submission, re-check the official AAAI author kit for the target year.
