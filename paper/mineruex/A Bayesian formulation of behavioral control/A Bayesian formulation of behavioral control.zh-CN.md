## 摘要

无助感，即一种“世界不受行为控制”的信念，长期以来一直处于我们理解抑郁的核心位置，并深刻影响了认知理论、动物模型和行为治疗。然而，尽管这一概念极其重要，心理学和精神病学中至今仍没有一个被完全接受的“无助”或“行为控制”的定义，而工程学中的形式化处理似乎也只能捕捉这些直观概念的有限侧面。本文将可控性形式化为带有情感价环境之上的先验分布特征。我们考察这一控制概念与在此类环境中优化行为的强化学习方法之间的关系，并讨论那些表面上看似不适应的信念如何能够由规范性的推断过程产生。最后，我们将这些结果与抑郁及其动物模型联系起来进行讨论。

## 1. 引言

“控制”和“可控性”的概念长期以来一直是理解和经验建模焦虑与抑郁的核心（Abramson, Seligman, & Teasdale, 1978; Abramson et al., 1998; Maier & Watkins, 2005; Seligman & Maier, 1967; Willner, 1985b; Williams, 1992）。其核心假设是：个体的抑郁性行为（以及焦虑性行为）可以被理解为源自一种信念，即强化结果超出了他们的影响范围，因此奖赏更难被利用，惩罚也更难被避免。尽管这一观点受到过重要批评（例如 Blaney, 1977; Buchwald, Coyne, & Cole, 1978; Costello, 1978; Frazer & Morilak, 2005; Willner, 1986; Willner & Mitchell, 2003，第 2 章），无助概念的认知表述仍然是健康个体抑郁风险的有力预测因子（Alloy et al., 1999），并为认知行为疗法提供了重要理论基础，而认知行为疗法正是抑郁的主要非药物治疗方式之一（Alloy & Abramson, 1982; Alloy et al., 1999; Beck, 1967; Beck, 1987; Beck, Rush, Shaw, & Emery, 1979; Williams, 1992）。此外，在动物模型中，对可控性的实验操控，例如习得性无助（LH）、慢性轻度应激（CMS）、悬尾实验和强迫游泳实验（Anisman & Matheson, 2005; Willner, 1985b, 1995, 1997; Willner & Mitchell, 2002, 2003），也是现代抑郁研究的关键组成部分，并构成抗抑郁药测试的重要平台。

在这些动物模型中，健康个体首先暴露于某种他们无法控制的环境性强化，例如电击。随后，在一个泛化任务中测量这种经历对他们在其他环境中的行为影响，例如观察那些经历了不可控电击的动物学习逃避反应的速度。动物模型实际上隐含提出了关于心理过程的至少两类基本主张：

1. 统计性主张：动物在新环境中的行为会受到既有知识或预期的影响。
2. 病因学主张：动物会学习这些先验信念（它们可能是不适应性的），并把这些信念泛化出去。也就是说，有过过去不可控电击经历的动物，会预期在新情境中的电击同样不可控；正是由于这种信念，它们才在新环境中也不再尝试控制电击（Maier & Watkins, 2005）。

与病理之间的确切关联值得细致讨论。关键在于，这些心理过程在健康个体中被假定为正常运作。也就是说，动物被认为能够正确评估自己拥有多大程度的控制，并把这类知识恰当地泛化到新环境中，因此后续学习迟缓是规范性的结果。若这些模型确实捕捉到了抑郁行为表型的重要方面，那么在人类中便存在两条通向精神障碍的路径，而这两条路径都建立在不适应性的先验信念之上。其一，功能失调可能是完全规范性推断的一个极端侧面。换言之，负性事件的经历，尤其是带有不可避免性和不可控性感知的事件，可能在抑郁障碍的形成中发挥因果作用（Blaney, 1977; Beck et al., 1979; Kendler et al., 1995; Kendler, Gardner, & Prescott, 2002; Kendler, Hettema, Butera, Gardner, & Prescott, 2003; Miller & Seligman, 1975; Peterson, Maier, & Seligman, 1993）。其二，推断过程本身可能是正常的，但它所依赖的先验分布却过于悲观或负性。这为一个可能性提供了空间：更长时间尺度上获得、甚至由遗传编码的先验信息，如何与具体环境中的信息相互作用，而这正是近期一些关于抑郁遗传因素的重要理论所提出的（Caspi et al., 2003）。

本文从正式的、规范性的、贝叶斯强化学习（RL）角度，对这些心理过程给出计算刻画。我们将可控性解释为决策问题先验分布的某些特征。我们考虑这样一种情形：个体面对一段较短的决策序列，但他们并不确定世界的精确结构，因此也不确定自己行动的后果。在这种情况下，个体应当使用信息性先验，这些先验捕捉了关于可控性的统计特征，从而辅助决策。我们将具体展示，这些先验如何影响个体对自己行动所能达成的事情的预期，包括环境状态之间的转移，以及奖赏和惩罚的获得。

在工程学语言中，一个系统若存在某个指令序列，可以将其从任意状态带到任意其他状态，那么该系统大致可称为“可控”。然而，这一定义与 LH 等范式中的心理学概念只有松散联系，因此我们的第一个任务（第 2 节）就是建立一种更适合的形式化。我们从一种接近原始文献中的看法出发（Maier & Seligman, 1976），即动作与结果之间映射的偶联性、可靠性或熵。我们将说明这一概念的优点，并以此为基础提出另外两种更整体的控制概念，它们分别关注：有多少结果可以被可靠实现，以及这些结果是否足够“有价值”。本文重点在于发展这些概念及其在强化学习情境中的后果，而非对动物或人类抑郁实验数据进行细致比对。第 3 节展示关于控制的先验信念在强化学习中的后果，第 4 节将其应用到习得性无助，第 5 节则讨论若干相关问题，包括目标导向选择与习惯性选择之间的区分、多巴胺的作用，以及奖赏与惩罚之间的对称性。

## 2. 控制的概念

我们首先概述三种主要的控制概念，它们为解释行为数据提供了层层递进、愈发具体的可能性。这些概念彼此建立在前一个概念之上，逐步捕捉在不同情境下“控制”一词所可能包含的额外特定含义。第一种概念捕捉结果的可靠性；第二种概念捕捉任一结果在多大程度上可以被可靠实现；第三种则与某些特定“理想”结果是否能够被可靠实现有关。所有数学细节均见在线补充材料。

为简化起见，我们考虑一类极简环境，一个很好的例子是“运行不完美的自动售货机”。环境中只有一个状态，有若干个离散动作 $a$（例如按下售货机上的某个按钮），每个动作都可能产生若干个结果（例如不同的巧克力棒）。动作到结果的映射可能是概率性的，且在最初对个体未知（按钮没有标签），尽管个体可能已经积累了少量试验经验。不过，个体被假定知道各个结果的效用，也就是每根巧克力棒的价值。我们进一步设定个体还可以再做一串长度为 $D$ 的动作，并特别关注这样一个事实：为了让后面的选择更有效，个体可能在前几次选择中用来探索那些尚不完全清楚的动作。

**熵。** 第一种也是最基本的控制概念（Maier & Seligman, 1976 提出，并构成“抑郁现实主义”工作的基础）与每个动作可能产生的不同结果的广度或分散度有关（图 1A）。我们用结果熵来形式化它。若一个动作对应各个结果的概率为 $p_o$，则其结果分布的熵为：

$$
\mathcal { H } = - \sum _ { o } p _ { o } \log ( p _ { o } ) .
$$

如果一个动作更确定地通向单一结果，也就是熵更低，我们就认为它具有更高的控制；反之，如果它以相近概率通向许多不同结果，也就是熵更高，则控制较低。以售货机为例，如果我们每次按同一个按钮几乎总拿到同一根巧克力棒，那就比按一次按钮却可能拿到很多不同东西更“可控”。为了方便起见，我们用“可能结果的数量”（结果集合大小）作为熵的一个合适代理量。

**可达结果。** 熵这一指标是把动作彼此孤立来看待的。这在存在多个动作时会出现反常，例如：如果所有可选动作都以确定性的方式导向同一个结果，它也会被赋予很高的控制水平（图 1B 左）。在售货机场景中，这就意味着所有按钮都会出同一根巧克力棒。于是，我们把控制扩展为：不仅要看动作本身的结果是否稳定，还要看环境中的各个可能结果是否都能被可靠实现。结合前面的定义，若一个主体的所有动作都满足：(i) 结果熵低；(ii) 不同动作导向不同结果，那么我们说它有更高的控制。图 1B 展示了这一概念。它已经比较接近工程学中的标准可控性概念。

**可达奖励。** 上述两种概念都不区分结果本身的“好坏”。但请考虑一种情况：主体只有一个最主要的需要，而环境中的动作能确定性地产生几乎所有其他结果，唯独不能产生满足这一需要的结果。例如，我们想从售货机里拿到某一根特定巧克力棒，但按钮只会出别的糖果。更贴切地说，在标准 LH 范式中，两组关键被试（主人组和配对组）遭受完全相同的电击，但只有主人组能够控制电击持续多久。配对组老鼠通常也能做出很多动作，但其中没有任何一个能决定电击何时结束。因此，我们将可控强化 $\chi$ 定义为：那些能够由某个动作所控制的结果中所能获得的奖励占总奖励的比例（见图 1C）。这一第三概念重新框定了前两个概念。它不再对所有结果一视同仁，而是让高相对奖励的结果比低奖励结果权重更大。为方便起见，我们的形式化只直接处理奖励。对于惩罚，我们采用一个数学技巧：把所有动作都与“最坏可能结果”比较，从而使它们都显得中性或有益，形成一种安全信号。尽管奖赏与惩罚在行为后果上存在重要不对称性，习得性无助却似乎能在奖赏与惩罚之间泛化，这一点我们会在文末讨论。

**图 1. 控制的概念。** 这些图给出了在选择某个动作条件下，对结果的概率分布。(A) 熵：如果一个动作会随机地产生许多概率相近的结果（左），那么控制较少；如果它只可能导向少数几个结果（右），那么控制较多。(B) 可控可达结果的比例：当动作不止一个时，不同动作结果之间的关系就很重要。在这些柱状图中，每一列代表一个动作的结果分布。共有四个动作，每个动作有四种结果。看最左边的图，所有动作都倾向于导向同一个结果，这就像不管按哪个按钮，售货机大多数时候都吐出同一根巧克力棒。中间的图中，每个动作都倾向于导向不同结果。最右边的图则是介于二者之间的情形：售货机可以比较稳定地产生广告里四种结果中的三种。结果 3 确实会出现，但没有任何一个动作比其他动作更倾向于产生它。控制与环境中所有结果能否被可靠实现的程度相对应。(C) 可控可达强化的比例：如果那些情感上特别显著的结果处于行为控制之下，控制就最多。红色柱表示各结果对应的强化值。左边情形中，所有强化都集中在对所有动作而言最可能出现的结果上；也就是说，售货机所有按钮都倾向于给出我们最想要的那根巧克力棒。中间图中，有一个按钮更倾向于给出目标巧克力棒，而其他按钮则倾向于给出不带奖励的结果；这两种情况下对奖励都有较大的控制。然而，如果强化分布像右图红柱所示，那么除了承载奖励的那个结果外，其余结果都能被可靠唤起；我们最想要的那根巧克力棒最可能来自一个会随机产生所有结果的动作。在这种情况下，可控可达奖励很少。

## 2.1. 泛化

与标准 LH 实验一样，我们假定个体会通过执行动作并观察结果来探索环境，并利用这些信息推断自己拥有多大程度的控制，也就是对上述控制量形成后验分布。关键问题在于：这些知识在将来如何泛化或迁移到新环境中的新决策问题上。

因此，这里其实有两个彼此独立的问题。第一是先验分布的形状问题，对此我们刚才提出了三种方案。第二是不同环境之间如何关联，更具体地说，它们在多大程度上共享相同的可控性。图 2 展示了两个极端。图 A 中，环境之间并不共享可控性，一个环境中的知识完全不会泛化到另一个环境；图 B 则相反，所有环境完全一样，在一个环境中获得的信息可以无差错地应用到其他环境。

这两个极端，其实都可以追溯到人类抑郁研究和动物模型中的一个区分。跨越多种环境真正保持恒定的因素，其实只有行动者本身。因此，假设不同环境共享某些控制特征，可能对应于一种信念：真正决定控制有多少的，是行动者自身。在人类文献中，这被称为“控制点”（locus of control），并被视为归因风格的一部分。在经典 LH 实验与 CMS 之间，也可以做出类似区分。后者中，动物暴露在许多环境里，每个环境都带有压力，只是程度较轻。这很可能会促使动物把“控制缺失”更多归因为一种内部的、可泛化的变量，而不是环境之间的外在差异。

**图 2. 泛化的两个极端。** (A) 无泛化：每个环境 $E$ 都有自己独立的控制设定 $c$。(B) 完全泛化：所有环境共享同一个控制设定 $c$。

需要强调的是，动物中观察到的这些行为，以及其在人类中的可能对应物，都依赖于相当强的泛化。因此，本文接下来集中讨论不同环境之间共享控制先验的情形。

## 3. 关于控制的先验信念的后果

对于一个决策问题的先验预期，会深刻影响规范性行动选择的三个关键方面：探索，也就是反复尝试不同动作的倾向；期望奖励，也就是个体可以期待获得的效用；以及动作之间的激励对比，也就是个体预期会在不同选择之间形成多大程度的偏好差异。本节将说明，不同形式和不同程度的控制会如何影响这些方面；定量细节见补充材料。

为了具体说明，我们继续使用售货机场景。设想售货机上有两个没有标签的按钮，$k$ 和 $u$，每个按钮都有 $L = 5$ 个可能结果，结果 $o \in \{1,\ldots,5\}$ 的奖励为 $R_o = o$。假设我们已经按过“已知按钮” $k$ 三次，其结果如图 3A 内嵌图所示，但除此之外并不了解它；而“未知按钮” $u$ 从未被按过。对于它的结果分布，我们除了先验分布之外一无所知，因此它是平的。如果主体只剩一次选择，那么最优策略就是按那个期望奖励最高的按钮。动作 $a_k$（按按钮 $k$）的期望奖励就是 $\sum_o c_o^{a_k} R_o$，其中 $c_o^{a_k}$ 表示执行动作 $a_k$ 后观察到结果 $o$ 的概率。由于真实的结果概率 $c_o^{a_k}$ 未知，因此无法直接计算真实期望奖励。然而，给定观测数据（这里所谓的充分统计量只是结果频数计数 $\mathbf{n}^{a_k}$），我们可以用贝叶斯法则把观测与先验结合起来，从而得到关于结果概率的后验分布：

$$
p ( \mathbf { c } ^ { a _ { k } } | \mathbf { n } ^ { a _ { k } } ) \propto p ( \mathbf { n } ^ { a _ { k } } | \mathbf { c } ^ { a _ { k } } ) p ( \mathbf { c } ^ { a _ { k } } )\tag{1}
$$

这里，第一个因子 $p(\mathbf{n}^{a_k}|\mathbf{c}^{a_k})$ 是：在某个真实但未知的离散结果分布 $\mathbf{c}^{a_k}$ 下，观察到与动作相关的结果计数 $\mathbf{n}^{a_k}$ 的似然。第二个因子 $p(\mathbf{c}^{a_k})$ 则是关于什么样的结果分布更可能出现的先验信念。我们认为，不同类型的控制正是通过这个因子实现的。这里的控制只影响“预测哪些结果会出现”，而不影响“这些结果各自具有多大回报”。对于 $a_u$ 和 $\mathbf{c}^{a_u}$ 来说，完全同理，只不过 $\mathbf{n}^{a_u} = \mathbf{0}$。

从这些后验分布出发，我们通过对所有可能概率分布求平均，得到所需的各种量。这包括期望奖励

$$
Q ( a _ { k } ) = \sum _ { o } R _ { o } \left[ \int _ { { \bf c } ^ { a _ { k } } } d { \bf c } ^ { a _ { k } } p ( { \bf c } ^ { a _ { k } } | { \bf n } ^ { a _ { k } } ) { \bf c } ^ { a _ { k } } \right] _ { o }
$$

以及下一次选择该动作时结果的预测分布 $p(n_{D+1}|\mathbf{n})$。这两者都取决于一组参数 $h$，它们决定了关于环境可控程度的先验信念。

**图 3. 关于结果熵先验信念对 Q 值与探索的影响。** 例子中售货机有两个动作和五种结果。(A) 如果先验认为结果分布熵较低，那么预测会更集中在已观察到的结果上。售货机的已知按钮 `k` 被按了 3 次，观察到结果 2 一次、结果 4 两次。深色柱表示把这些观测与“结果分布较窄”的高控制先验结合后的预测，浅灰色柱表示与“结果分布较宽”的低控制先验结合后的预测。(B) 在高控制先验下，即时期望奖励略高于低控制先验，因为观测结果略有偏斜，而低控制先验会削弱这种偏斜。比较对象是未知按钮 `u`，其期望值为 $Q(a_u)=3$。(C) 探索奖励：当还剩 $D=1,2,3$ 次选择时，已知按钮和未知按钮的 Q 值差异。只有在高控制先验下，探索奖励才明显出现。(D) 高控制先验与低控制先验下每个动作 Q 值的差异，这是对图 C 数据的另一种展示方式。

## 3.1. 单个动作的结果熵

结果熵决定了在反复按售货机某一个按钮时，会吐出多少种不同的巧克力棒。图 3A 给出了高控制与低控制条件下的预测分布示例（另见补充材料公式 (10)）。如果主体强烈相信自己拥有广泛控制，那么其先验 $p(\mathbf{c}^{a_k})$ 会使低熵分布本身更有可能出现；换句话说，要让主体相信自己其实没有控制，数据必须给出很强证据。因此，在高控制先验下，预测概率质量 $p(n_{D+1}|\mathbf{n})$ 会高度集中在已经观察到的结果上；而在低控制先验下，预测分布则更宽。图 3B 到图 3D 展示了这些预测带来的后果。

### 3.1.1. 探索、激励对比与平均奖励

对于尚未探索的动作 $a_u$，其预测分布是平的，因此只执行一次该动作的期望价值 $Q^1(a_u)$ 为 3。对于已知动作，结果 4（奖励 4）被观察到两次，而结果 2（奖励 2）只观察到一次。因此，在高控制和低控制先验下，动作 $a_k$ 的期望价值 $Q^1(a_k)$ 都高于 $a_u$，只是这种优势在高控制情形下更强（图 3B）。

然而，如果剩余选择次数不止一次，那么尝试未知按钮 $a_u$ 以确定其效用是否可能超过 $a_k$ 就可能是值得的。若真如此，后续选择中就可以利用这一更优选项。围绕 $a_u$ 的这种不确定性的价值，恰好就是它潜在收益的来源，从而推动了探索。在强化学习中，这被称为“探索奖励”（exploration bonus）。

这一点之所以可计算，是因为我们处理的问题规模很小。对于更现实的最优探索问题，只有在一种非常受限的情形下我们才知道最优策略，那就是通过所谓 Gittins 指数来表示。不过，Gittins 指数在结构上较脆弱，一般情形下并不适用，因此通常还是需要近似探索奖励。

探索奖励的大小取决于控制程度。为了看清这一点，设想按钮 $a_u$ 被按下后产生了结果 5，也就是最好的巧克力棒。在高控制先验下，预测分布现在会强烈集中在结果 5 上，因此下一次还应继续选择按钮 $a_u$。但在低控制先验下，这个单次结果对预测分布的影响很小，主体会把得到结果 5 解释为纯粹偶然，不会认为再次按下 $a_u$ 还能重复这一幸运事件。其后果是：$a_k$ 仍然看起来更优，从而抑制探索。

因此，在高控制先验下，不仅那些导向好结果的动作会被积极利用，导向负结果的动作也会被积极回避；同时，由于未来可以利用这些信息，不确定性本身也使探索变得值得。而在低控制先验下，结果对动作选择的偏置更弱，未来可利用性的缺失也削弱了探索奖励。图 3C 给出了在两种控制情形下、还剩 $D=1,2,3$ 次动作选择时，$Q^D(a_u)-Q^D(a_k)$ 的差异。对于高控制，这个差值为正，体现了不确定性奖励；且其绝对大小也更大。换句话说，如图 3D 所示，在高控制先验下，不同动作之间的激励对比更强。此外，由于奖励更容易被利用、惩罚更容易被避免，高控制先验下的总体期望奖励总是不低于低控制先验。

### 3.1.2. 泛化

接下来的关键问题是：对环境控制水平的准确知识是否真的有信息价值？换句话说，知道环境到底有多可控，是否会带来更好的行为？如果答案是肯定的，那么在共享可控性的环境之间进行泛化就可能是有利的。上一节已经说明，如果主体低估了环境的可控性，那么一些本可利用的动作就会被错过。图 4 考察了相反情况，也就是高估环境中动作的可控性会产生什么后果。结果显示，在这种意义下，只有低估控制才会真正造成问题；平均来看，高估控制并不会损害表现，因为如果结果本身具有高熵，那么任何动作几乎都一样好，或者一样差。总之，在这一层次上，最大化奖励的行为来自一个固定的“低结果熵”假设；知道环境的真实可控程度，并不会转化为更高的平均奖励。对下面更复杂的控制概念来说，结论就不再如此。

## 3.2. 可控可达结果

当“控制”的概念不仅包含“每个动作各自的结果分布是否有峰值”，还包含“不同动作的峰值分别位于哪里”时，推断真实控制水平并将其泛化到新环境就变得有利了。第二种控制概念，也就是“可控可达结果”，正是做这件事。

在我们的简化表述中，所有动作的联合结果概率先验 $p(\mathbf{C})$ 通过一个辅助二值矩阵 $M$ 来定义，其中第 $ij$ 个元素决定结果 $i$ 是否是动作 $j$ 的“可控可达结果”。每个动作最多只能有一个可控可达结果；因此，如果矩阵 $M$ 的某一列在某个结果位置上为 1，那么该动作的结果概率分布就会在该结果处出现峰值。带有单个 1 的列的总数，恰好就是具有某个可控可达结果的动作数量；而这些列所对应的不同结果的数量 $|M|$，就是可控可达结果的数量。如果共有 $L$ 个可能动作，那么 $|M|=L$ 就表示“可控可达结果的比例”为 1，也就是每个可能结果至少都是某个动作的可控后果。矩阵 $M$ 因此把控制的底层结构形式化了。

由 $M$ 生成的矩阵 $C$ 决定了每个动作产生每个结果的实际概率。当 $\mathbf{M}_{ij}=1$ 时，$\mathbf{C}_{ij}=c$，而对所有 $k \neq i$，有 $\mathbf{C}_{kj}=(1-c)/(L-1)$。于是，当 $|M| \to L$ 且 $c \to 1$ 时，不同动作的结果分布会相互分离。这里的 $c$ 捕捉的就是前面讨论过的“熵式控制”效应。如果动作 $j$ 没有任何可控可达结果，那么对所有 $i$ 都有 $\mathbf{C}_{ij}=1/L$，也就是达到最大熵。

**图 4. 环境可控性与主观信念不匹配。** 图中展示了在不同主观先验信念和不同环境可控性下，平均获得的强化量。只有低估环境可控性（在结果熵意义上），而非高估，才会带来不良后果。当环境高度随机时，主体行为以及其先验信念几乎没有影响。

图 5 进一步说明：在这种更丰富的控制定义下，把对控制的准确估计泛化到新环境中确实是有利的。例子设置很简单：五个可能结果中只有两个（1 和 5）带来奖励，分别为 0.3 和 0.7，而其中只有较差的那个奖励 0.3 是可控可达的。具体地，辅助矩阵 $M$、五个结果的结果分布 ${\bf C}=\{{\bf c}^a\}_{a=1}^5$、奖励向量 $R$，以及每个动作单独执行一次时对应的真实期望结果向量 $Q$ 如原文公式 (2) 所示。

如前所述，矩阵 $M$ 同时指示“结果分布的峰值在哪里”以及“是否存在这样的峰值”；而 $C$ 则给出实际的结果分布。此时 $|M|=1$。由于动作 1 大约有 80% 的概率可控地产生结果 1，所以它是最优动作，尽管它并不会稳定地产生全局最优的那个结果。

参数 $c$ 的存在意味着，这一可控性概念继承了以单动作结果熵为基础的控制概念的大多数性质。但与纯熵情形不同的是，若主体错误地假设“有太多动作都是可控可达的”，就可能产生实际损害。图 5 明确展示了这一点。

研究者通过随机选择动作生成不同数量的观测，并依据真实分布 $C$ 采样结果。随后，在给定这些数据以及不同先验的条件下，计算后验分布和预测分布。先验分布允许 $|M|$ 个可控结果，也就是只允许那些与某个 $|M|$ 值一致的矩阵 $C$。图中给出了在“两次额外选择中的第一次”里，各动作被选中的频率；也就是在经验 $\mathbf{N}$ 给定时，满足 $Q^2(a_i|\mathbf{N}) > Q^2(a_k|\mathbf{N})$ 的比例。这里使用 $Q^2$ 是为了把探索奖励也包含进去。

图 5A 到 5D 表明：如果主体正确地假设“只有一个结果是可控可达的”，它就会利用动作 1。随着假设中可控可达结果的数量增加，探索会更持久。极端情况下，如果主体认为所有结果都可控可达，那么尽管动作 1 是最优动作，它最终反而会被回避。随着用于推断预测概率的先验观测数增加，这一模式会更加清晰，但在观测很少时就已经可以看到。只要最高奖励并不可利用，那么“假设比真实情况更多的结果是可控可达的”就会导致更持久的探索，并阻碍足够的利用。因此，可控可达结果的比例确实是环境的一个有信息价值的特征，所以将其泛化是合理的。

**图 5. 过度控制的幻觉可能有害。** 图中各面板显示：在五个可选动作中，每个动作被选择的比例如何随先验观测数量以及关于“可控可达结果数量”的先验信念而变化。随着观测增加，当先验正确地假设 $|M|=1$ 时，最优动作 1 被利用得最多；而当先验错误地假设所有结果都必须可控（$|M|=5$）时，最优动作 1 反而被回避。

## 3.3. 可控可达奖励

到目前为止，我们已经讨论了与控制有关的结果分布的两个方面：结果熵，它更接近心理学中的想法；以及结果可达性，它更接近工程学中的可控性概念。最后还有一个关键成分，那就是强化。真正重要的，或许并不是“可控结果的粗略数量”，而是“对那些携带大部分强化的结果的控制”。在动物模型里，控制通常被定义为：是否存在某个动作，可以达成一个理想目标。同样，在人类研究中，无助也通常表现为对人际关系或工作中的高价值回报缺乏控制。

因此，我们转向第三种也是最后一种控制概念：环境中可控可达强化所占的比例（图 1C）。在高度抽象化的环境模型中，我们使用变量 $\chi$（补充材料公式 (20)）来刻画：有多少比例的强化是通过可控可达结果获得的。例如，在图 5 的情形中，由于总强化中只有 0.3 是通过可控可达结果获得的，而控制程度为 $C_{11}=0.8$，所以 $\chi=0.24$。

这一控制概念同样继承了前两种概念的主要性质，但由于它直接聚焦于强化，因此在心理学上更细致。图 6 展示了 $\chi$ 与强化结构对预测分布的影响。在该例中，$|A|=5$ 个动作中的每一个都已经被执行过 3 次，而且每次都导向结果 $o=a$，也就是存在充分证据表明“控制是完美的”。图 6D 和 6E 使用的是图 B 的奖励结构，其中所有结果都带有某种奖励，只是大小不同。在图 D 中，$\chi=1$，因此只有那些每列都恰好有一个 1 的矩阵 $M$，以及与之对应的低熵结果概率向量，才会对预测作出贡献。总体上，所有动作都会得到非常低熵的预测分布，因为所有动作都带来奖励。然而，当 $\chi$ 被设为 0 时，预测分布会发生变化。此时，只要动作会产生带奖励的结果，先验就暗示它不会以低熵方式做到这一点。因此，由于所有动作都通向某些奖励，所有动作的结果分布熵都会升高；其中最明显的，是通向最高奖励的动作 1。图 6F 展示了更极端的版本：只有动作 1 会导向被强化的结果。此时，除了那个产生奖励的动作外，所有动作都被预测为确定性地产生结果；而唯一带来奖励的动作则具有高熵。也就是说，可控奖励比例这一概念使我们能够捕捉无助中“针对强化”的那一面。

**图 6. 对强化敏感的控制。** (A) 对每个动作 $a$，都观测到了 3 次结果 $o=a$。(B) 图 D 和 E 使用的各结果奖励比例；这里所有结果、因而所有动作都携带相当可观的强化。(C) 图 F 使用的奖励比例：只有一个结果携带全部奖励。(D)-(F) 推断得到的动作-结果矩阵。由于该树是由重复选择构成的，它们同时也是推断得到的转移矩阵。(D) 当假定图 B 中有很大比例的奖励是可控可达的（$\chi=1$）时，对所有动作都恢复出低熵预测分布。(E) 但当 $\chi=0$ 时，所有预测分布都变为高熵，而且某动作对应结果的奖励越高，这种效应越明显。这里的奖励结构仍然来自图 B。(F) 如果采用图 C 的更极端奖励结构并令 $\chi=0$，那么那些不导向奖励的动作会对应低熵预测分布，而唯一导向环境中唯一奖励的那个动作则对应高熵预测分布。

## 4. 习得性无助

接下来我们考察，对奖励敏感的控制概念如何解释 LH 的主要特征。标准实验设置如图 7 所示，包括主人组、配对组和控制组。在基于电击的无助训练中，主人组和配对组老鼠都在同一个环境中遭遇不可预测时点开始的电击；电击何时结束取决于主人组是否做出某个逃避动作，而不取决于配对组做了什么。我们假定个体会据此提取出关于控制程度 $\chi$ 的一个分布，然后在第二个环境中使用这一分布来形成预测。在第二个环境里，它们必须学习一个实际上完全可控的逃避反应，其中动作 1 会带来奖励 0，而其他动作都带来奖励 -1。第 3.1.2 节已经表明，通常只要控制参数设定正确，就会得到最优表现；这里我们再次说明：如果在第二环境中使用一个过小但却是“正确推断出来”的 $\chi$ 值，那么逃避训练将受到强烈影响。补充材料第 3 节展示了，如何从第一个环境中的过去观测 $N$ 推断出 $\chi$ 的最大似然估计，以及如何在导出预测分布时把过去观测作为关于 $\chi$ 的额外约束。正如上文所说，我们通过把结果与最坏可能情况比较，来把“避免惩罚”转换为“追求安全”：

$$
\tilde { R } _ { i } = R _ { i } - \operatorname* { m i n } _ { j } R _ { j } ,\tag{3}
$$

这一处理的合理性我们会在讨论中进一步说明。

图 8A 和 8B 展示了：在两个初始环境中观测之后，关于 $\chi$ 的后验分布。第一个环境提供了较高的可控可达强化（$\chi=0.9$），第二个只提供较低的可控可达强化（$\chi=0.1$）。在两种情况下，都是通过随机动作选择总共生成了 80 次观测，而后验分布都正确地峰化在高值或低值附近。随后，主体被转移到另一个环境，并再经历 80 个结果；此时每个动作 $a$ 都导向一个固定的、确定性的结果 $o=a$。使用从第一个环境中得到的先验，以及第二个环境中的观测，就可以为每个假设的 $\chi$ 值得到关于未来结果的预测分布，并对图 8A 和 8B 中的分布求平均。

图 8C 表明：如果使用来自图 8A 的低控制分布，预测将具有高熵；而图 8D 表明：如果使用来自图 8B 的高控制分布，预测则是低熵的。和前面一样，这些预测分布可以用来计算每个动作的 Q 值。图 8E 显示，在经历了“可控强化”之后，动作 1 的价值明显更高，而且动作间差异也更大；平均价值也更高。图 8F 进一步考察了这一点，它展示了随着动作 2 到 4 的电击强度变化，动作 1 与动作 2 之间的差异如何变化。正如预期，在经历可逃避电击之后，电击强度变化对 Q 值的影响要比经历不可逃避电击之后更大。最后，图 8G 和 8H 给出了动作选择概率随电击强度变化的情况。与动作 1 和动作 2 的 Q 值差异一样，选择概率差异在经历可控电击之后增长得更快。在对第二个可控测试环境进行了足够多的暴露后，各组之间的差异会消失，因为此时关于 $\chi$ 的学习仍在继续。

该模型复现了 Maier 与 Watkins（2005）报道的部分泛化结果：配对组动物即使偶尔一开始能正确逃避，也不会稳定偏向逃避。这里，主体起初并不知道各动作通向什么结果，因此会随机选择动作。即使后来得到了很强证据表明自己能够逃离电击，它们对逃避动作的偏好仍然很弱。该模型也复现了 Jackson、Maier 与 Rapaport（1978）的发现：提高电击强度会减轻 LH 的效应。设想逃避任务中的电击强度已经高到 5，以至于经历了可逃避电击的动物已经达到天花板水平；继续提高电击强度不会进一步增加主人组选择逃避的概率，但会提高经历了不可逃避电击的动物选择逃避的概率，因为这些动物仍然会受到实际 Q 值的影响。

**图 7. 习得性无助范式。** 三组老鼠依次接受两个任务。第一个任务中，老鼠暴露于可逃避或不可逃避的不可预测电击中。主人组老鼠面对的是可逃避电击：它们可以通过执行某个动作（通常是转动轮子）关闭电击。配对组暴露于与主人组完全相同的电击，也就是说，只有当主人组终止电击时，它们自己的电击才会终止，因此对它们而言电击是不可逃避的。第三组老鼠不接受电击。随后，三组老鼠都进入穿梭箱逃避任务。电击再次在随机时刻出现，老鼠必须移动到箱子的另一侧才能终止电击。只有配对组老鼠不能学会这一逃避反应。

**图 8. 急性强电击后的习得性无助。** 模拟方式是：先在一个环境中推断 $\chi$ 的分布，再将其作为第二个环境中的先验。在第一个环境中，除了结果 4 之外其余结果都是负性的；在第二个环境中，除了结果 1 之外其余结果都是负性的。(A) 在低控制（$\chi=0.1$）且呈现不可逃避电击（IS）的环境中，基于 80 次观测得到的关于可控可达强化的后验分布，质量集中在低值上。(B) 在高控制（$\chi=0.9$）且呈现可逃避电击（ES）的环境中，基于 80 次观测得到的后验分布。(C) 和 (D) 为测试环境中每个动作的结果预测分布。测试环境中每个动作都被观察到 20 次，且每次都得到 $o=a$，这为“完全控制”提供了很强证据；但是，如果使用图 A 的低控制先验，预测分布仍然具有高熵，而使用图 B 的高控制先验时，预测分布则是低熵的。(E) 测试环境中四个动作的 Q 值。经历不可控强化之后，最佳动作（动作 1）的期望回报低于经历可控强化之后；同时，各动作之间的差异也被压缩了。(F) 提高测试环境中的惩罚强度后，动作 1 相对其他动作的优势在“经历可控强化”后增加得更明显，而在“经历不可控强化”后增加得较弱。(G) 和 (H) 如果用 softmax 函数从 Q 值导出概率策略，那么在可控强化之后，随着强化强度上升，对动作 1 的偏好增长得比不可控强化之后更快。

## 5. 讨论

简单地说，控制通过让奖励更可利用、让惩罚更可避免，使世界变得更愉悦、更丰富，也更值得探索。我们提出的三种控制概念都具备这一点。这三种概念的力量逐层增强并彼此包容：可达结果比例依赖于结果熵的概念，而可达奖励概念则在纯粹“可达性”之上再增加了一个关键维度。

为了聚焦概念本身，我们使用的是较为贫乏且带有任意性的数学表述。例如，我们有时把结果分布写成均匀分布与 delta 函数的混合；也因此，在某种程度上用结果集合大小代替了 Shannon 熵。我们还只考虑了单状态情形。显然，这些都是非常激进的简化。不过，机器学习和贝叶斯强化学习文献中已有诸如相关 Dirichlet 过程之类的方法，可以用更灵活、更简洁、也更优雅的方式，在更一般的决策问题上表达关于先验 $p(c)$ 的类似思想。

即使在我们的简单表述里，不同控制概念之间也已经表现出显著差异。尤其是，关于泛化的结果表明：与其他两种概念不同，单独的结果熵本身并不是一个值得推断并投射到新环境中的量。当然，在受限环境类别中，它可能仍然是一些更有用量的简单代理。未来研究中，一个重要方向将是设计并使用更具体的行为测验，以区分并丰富这些不同的控制概念。

### 5.1. 习得性无助

我们用一个自己定义的量，即“环境所提供的可控可达强化比例”，对 LH 给出了定性解释。把这一量泛化到新环境中，可以解释配对组动物获得性的逃避缺陷；可以解释逃避缺陷为何会受到逃避阶段电击强度的影响；也能复现这样一个发现：即使动物已经得到了充分证据表明逃避动作确实能关闭电击，逃避缺陷仍然会持续。对于人类，我们的模型虽能对部分数据给出定性说明，但严格的行为学发现仍然极其匮乏，因此我们没有尝试去建模明确的主观判断。我们希望，这些形式化处理能使研究者在人的控制感与无助感方面采集到更精确的数据。

不过，这项工作仍然存在若干重大局限。也许最重要的一点不匹配在于：这些应激诱导的动物模型没有一个能真正摆脱焦虑成分；LH 本身甚至被认为更适合作为创伤后应激障碍，而不是抑郁的模型。哪怕只经历一次应激，也可能造成持久效应。Strekalova 等人的一项非常细致的研究发现，焦虑效应并不局限于 LH；即使是较温和的慢性轻度应激，也往往会产生快感缺失与焦虑的组合。从某种意义上说，食欲性 LH 是对该理论更精确的检验；否则，我们就需要把更完整的厌恶加工图景与本文这些先验结合起来。关于厌恶、以及厌恶加工和奖赏加工之间差异的计算观点虽然正在发展，但还不足以与本文这类分析直接结合。

其次，我们对泛化的处理过于简化。正如前文提到的，诸如动作平均效价的预测等方面，很可能也会发生泛化。最复杂的一点在于：巴甫洛夫式效应，尤其是与厌恶结果预测直接相关的退缩，可能会以不同程度扰动目标导向控制和习惯性控制，尤其是在厌恶领域。对这种相互作用的复杂性，研究才刚刚开始建立模型。在我们对 LH 的处理里，行动者实际上假定：同一个控制水平同时适用于两个环境。慢性轻度应激模型则指出了另一种方向：动物会依次暴露在多个轻度厌恶且不可控的环境中，只有在接触多个这样的环境之后，才会把这种性质泛化到新环境里。也就是说，动物起初会把环境相对分开，只有在有充分证据表明这些环境都共享一种厌恶性质时，才进行泛化。分层模型和混合模型可以容纳这种效应，它们也与人类中的“控制点”概念相关。

最后，我们这里只把最复杂的那种控制概念应用到了习得性无助上。其余两种控制概念是否也能解释 LH，取决于具体实验设置。例如，在慢性轻度应激中，动物虽然在许多环境里暴露于轻度厌恶刺激，但它们通常仍然可以在行为的其他方面自由活动并体验到可控强化。类似地，在习得性无助中，动物遭遇不可控电击的时间只有大约一小时，剩余时间都在家笼里。因此，总的来说，并不是它们“完全没有控制”，而是它们“对某些特定且情感上极其显著的结果没有控制”。正因为如此，最后这种“可控可达奖励”的概念才最适合用来理解 LH。

### 5.2. 抑郁

本文试图在规范性情感决策的框架下思考与精神病学相关的发现。我们导出的所有后果，都来自将标准的、规范性的、概率论与运筹学中的推断和最优行动选择原则应用到问题中，因此这提示我们：那些被称作“疾病模型”的现象，可能其实是主体对其环境中某类事件的最优反应。当然，这并不是什么全新的说法。事实上，在通过环境操纵让健康动物出现异常表现的精神障碍动物模型中，这一点早已隐含存在。需要指出的是，尽管动物模型和本文模型都说明：对环境偶联关系的正常反应在外表上可能看起来像抑郁，但它们并不声称、也无意声称这是通往疾病的唯一途径。正如引言中所说，即使在我们的模型内部，也完全可能是：个体在各个环境中的推断本身都是规范而正确的，但这种推断会被一个不规范的控制先验、或一个不规范的泛化倾向所干扰。这种不适应性的先验或泛化倾向，可能来自学习或回忆过去环境偶联时某些环节的失常，而其中一些方面很可能是由遗传编码的。进一步说，先验与环境之间的这类相互作用，也许能够捕捉所谓“基因—环境交互作用”的某些效应。

我们的解释只是对一种极其复杂、且尚未被充分理解的疾病中的某一部分做了非常简约的说明。我们最直接关注的动物模型，本身就已经与真实疾病相去甚远。因此，本文显然无意成为一个“抑郁模型”。相反，我们的目标是对一个在理解抑郁时至关重要的概念给出计算表述。正如引言里指出的，当前并不存在一个被接受的“控制”定义。我们希望，本文提出的这些定义，虽然只是粗略地建立在心理学、精神病学和系统控制中的一些思想之上，但能帮助我们用一种把强化学习与精神障碍联系起来的术语，来更精细地刻画这一现象。这样的表述应当有助于连接人类研究与动物研究，并推动对这些现象进行更具体的剖析，例如在纯行为设置中独立于这些先验信念的来源，去测量被试或患者的先验信念。

我们的观点也不同于另一些对精神障碍的计算性或规范性探索。我们并不认为抑郁是一种为了达成某个特定目标而进化出来的适应性手段；相反，抑郁行为来自环境特征与主体对这些特征的假设之间的不匹配。参数上的这种错配在既有工作中也曾被讨论过。再次强调，提出一条“规范性地通向抑郁”的路径，其主要价值在于：它使精神疾病的一部分问题可以进入强有力的形式分析之中。

### 5.3. 目标导向选择与习惯性选择

我们同样只给出了一个相当简化的图景，说明本文讨论的关键强化学习概念如何映射到情感控制的神经架构上，并由此与动物模型的行为神经科学相连接。简言之，二者之间确实存在比较直接的对应关系；但只有把它放到近期大量工作所强调的背景中，才能真正理解这种对应，而这些工作正是用来区分影响行动选择的不同算法结构和解剖结构，尤其是把习惯性或缓存式控制，与目标导向或前向模型式控制区分开来。不同的决策系统会以不同方式吸纳我们这里讨论的先验知识，因此也可能带来不同结果。

目标导向或基于模型的控制，需要构建环境模型，并通过一种树状搜索来找到最佳动作。由于我们正是把各种可控性概念表述为这类可学习模型之上的贝叶斯先验，因此它们可以直接影响目标导向决策。各种实验研究也暗示了这一点。此外，关于 LH 的人类文献也大量聚焦在有意识的、目标导向的行为与选择上；而近来的习得性无助神经生物学研究，则牵涉到了那些参与目标导向控制的脑区，这些脑区在人类抑郁以及正常高级认知功能中也都很重要。

与此相对，在习惯性控制中，动物被认为通过经验获得动作的缓存价值，从而在决策时不再需要树状搜索。习惯性学习并不使用环境的学习模型，因此也就无法轻易把关于这些模型的先验直接整合进缓存价值里。不过，这类先验的一些总体后果，例如环境的变异程度（与熵有关），甚至整体期望奖励，仍然会影响习惯性学习的进程。事实上，传统 LH 动物范式常常被放在习惯性而非目标导向的脉络下解释；相关方面也曾借助另一种心理学和精神病学概念“钝化”来讨论。所谓钝化，是指那些评估强化输入的基础系统，其反应性被压低。然而，与较可信的钝化模型不同，控制性的某些方面已知可以跨越强化价态进行泛化：暴露于不可逃避电击的老鼠会出现纯粹的食欲性学习缺陷，而暴露于不可控正性强化的老鼠同样会表现出逃避缺陷。因此，继续区分这些先验对不同决策系统分别产生什么影响，是非常迫切的工作。

### 5.4. 多巴胺

这项研究受到强化学习概念的一项重要启发：精神病学中那些显著的神经调质，在情感决策中究竟扮演什么角色。尤其是，多巴胺与抑郁之间存在相对紧密的联系，在某种意义上，这种神经调质的“紧张性水平”似乎是控制最自然的神经生物学载体。躁狂往往表现为控制妄想，并常用 DA 拮抗剂治疗。需要注意的是，多巴胺也曾被联系到“认知控制”，但那是一个与本文完全不同的“控制”意义。紧张性 DA 的升高既会增强特定动机驱力，也会增加总体行动。

Niv、Daw 与 Dayan 等人对其中一些效应给出了详细的定量解释。他们提出，紧张性 DA 报告的是单位时间内执行动作所期望获得的平均奖励，而这会构成一种机会成本，惩罚懒散并决定合适的反应活力。这一观点与本文的“可控可达奖励”表述相关，因为随着 $\chi \to 1$，动作越来越值得投入努力。事实上，有一些证据表明，紧张性 DA 不仅会被奖励增强，也会被“可控的惩罚”增强，而二者都需要激发适当行动。若要检验控制与多巴胺之间是否真有关联，一个很好的试金石是在“不可控奖励”的情境下测量紧张性 DA 水平。

在抑郁语境中，这一观点预测：动机缺陷会与“缺乏控制”的先验预期相关。重度抑郁患者似乎的确同时具有动机缺陷和无助感，但在这个问题上仍需要更具体的实验检验。

不过，这显然并非全部故事。我们已经指出，可控性是目标导向系统中的一个复杂构造；而多巴胺则更紧密地与食欲性的习惯控制相关。因此，它是否能够在与价态无关的情况下，表征像“可控可达强化”这样的变量，是值得怀疑的。此外，我们目前仍缺乏一个在描述上足够充分的模型，用来解释一般动机如何影响目标导向控制。

### 5.5. 奖赏与惩罚之间的对称性

推动我们提出这一可控性表述的一个特殊动机，是这样一种观察：暴露于不可控强化之后，其影响会跨越强化价态发生泛化。这一点尤其重要，因为奖赏加工与惩罚加工在神经生物学基础上差异巨大。这也是 LH 中一个无法由“完全没有控制概念的习惯机制”直接解释的关键方面，因为镇痛和奖励敏感性下降之间并没有直接联系。为了说明 LH 中所观察到的那种“钝化的对称性”，我们把可控可达强化表述成一个与价态无关的量：它只衡量环境中全部强化中有多少规范化比例是可用的。

在这些抑郁模型中，若没有那些直接评估目标导向学习的实验，例如强化物贬值实验，那么“对强化的行为性不敏感，并且这种不敏感在价态上是对称的”，似乎就是最强的证据，表明这里涉及了本文提出的那种目标导向控制概念。遗憾的是，人类抑郁方面的数据仍不足以支持明确结论。一些研究在原初强化敏感性上报告了对称效应，例如观看情绪影片时的生理反应，但这并不能直接说明目标导向系统的问题。另一方面，问卷数据似乎显示：抑郁者对惩罚的感知敏感性更高、对奖励的感知敏感性更低，但这些数据同时受到主观报告与原初敏感性变化的混杂影响。

在人类抑郁中，认知理论、LH 理论以及绝望理论，虽然不能直接按本文的行为强化学习术语来解释，但都认为“控制感下降”是抑郁的核心，而且这种下降同时适用于正性和负性事件。尤其是，抑郁个体通常会把正性事件归因于运气，把负性事件归因于自己无法触及的稳定原因。这意味着他们既无法利用正性事件，也无法避免负性事件，而这恰恰正是本文所讨论的那种普遍控制缺失的预期后果。

## 6. 结论

总之，我们从环境所提供结果之先验分布的特征出发，发展出三种关于可控性的形式化表述。在假设主体会从一组环境中推断控制程度，并将其泛化到其他环境的前提下，我们展示了这些表述如何能够定性地捕捉抑郁动物模型中的许多方面，而可控性被认为在这一病症中发挥着重要作用。我们希望，这些精确的形式化能为患者现象的澄清与分类提供新的基础。

# 在线补充材料译文

## 行为控制的数学表述

这里我们给出各类控制模型的数学细节：结果熵、可控可达结果比例，以及可控可达强化比例。

总体设定如下：环境被假定具有某个特定的控制水平，也就是说，观测的似然会由某个恰当定义的控制参数来参数化。生物体在一个或少数几个训练环境中收集观测，并据此推断训练环境中控制参数设定的后验分布。随后，生物体被转移到测试环境，只接触有限数量的观测。它们会把对测试环境控制水平的先验预期（以某种适当方式从训练环境中的控制后验导出）与测试环境中的观测似然结合起来，从而得到未来观测的预测分布。测试环境中的动作选择，则依据这些结果的预测概率来完成。

## 1. 将控制视为条件熵 / 结果集合大小

第一种、也是最基本的控制概念，是“在单个动作条件下结果概率分布的熵”（Maier and Seligman, 1976; Overmier et al., 1980; Gibbon et al., 1974）。它与独立动作的结果集合大小密切相关，也就是对某一动作而言可能被观察到的结果数量。结果集合大小与条件熵有关，但在分析上更方便。我们在这里紧密跟随 Friedman and Singer（1999）以及 Dearden 等（1998, 1999）的工作。问题设置如下：给定若干动作—结果观测，以及一个关于“可能会观察到多少种不同结果”的先验信念，最优动作选择应当是什么？而最优动作选择，又将由对该动作可能产生哪些结果的预测所决定。

先考虑一个单一动作，它有 $L$ 个可能结果。令 $X$ 为这些结果的一个无序子集，$|X|$ 为该集合的基数，也就是集合中不同元素的个数。例如，对于子集 $X=\{1,2,L\}$（假设 $L>2$），有 $|X|=3$。在总共 $L$ 个结果中，给定某个集合大小时，这样的集合共有 ${\binom{L}{|X|}} = L!/(|X|!(L-|X|)!)$ 个。我们现在对结果集合大小 $|X|$ 放置一个先验分布 $p(|X|)$，也就是对“某一动作预期会涉及多少种不同结果”放置先验，并假设所有基数相同的集合具有相同概率。于是就得到关于集合的先验：

$$
\begin{array} { c c l } { { p ( X ) } } & { { = } } & { { { \binom { L } { | X | } } ^ { - 1 } p ( | X | ) } } \end{array}\tag{1}
$$

进一步，把式 (1) 中的集合大小先验 $p(|X|)$ 参数化为带参数 $\zeta$ 的截断几何分布：

$$
\begin{array} { r l r } { p ( | X | | \zeta ) } & { = } & { \left\{ \begin{array} { l l } { 1 / L } & { \quad \mathrm { i f } \quad \quad \zeta = 1 } \\ { \zeta ^ { | X | - 1 } \frac { 1 - \zeta } { 1 - \zeta ^ { L } } \quad } & { \quad \mathrm { e l s e } } \end{array} \right. } \end{array}\tag{2}
$$

直观地说，当 $\zeta$ 很小时，几乎只允许集合大小为 1；当 $\zeta$ 很大时，则只有接近 $L$ 的集合大小会被保留。因此，参数 $\zeta$ 决定了结果集合大小，也就是本小节中我们对“控制”的参数化方式。

为了只呈现“结果集合大小先验”的纯效应，我们还需要把该集合内部的实际概率分布积分掉。令 $\mathbf{c}$ 表示一个动作的结果概率向量，即观察到结果 $i$ 的概率为 $c_i$；若结果 $i$ 被观察到 $n_i$ 次，则其似然是一个多项分布：

$$
\begin{array} { r c l } { p ( { \bf n } | { \bf c } ) } & { = } & { \displaystyle \frac { ( \sum _ { i } n _ { i } ) ! } { \prod _ { i } n _ { i } ! } \prod _ { i } c _ { i } ^ { n _ { i } } } \end{array}\tag{3}
$$

现在可以在多项分布的结果概率向量 $\mathbf{c}$ 上放一个由结果集合 $X$ 参数化的 Dirichlet 先验：

$$
p ( \mathbf { c } | X , \alpha ) = \frac { \Gamma ( | X | \alpha ) } { \prod _ { i \in X } \Gamma ( \alpha ) } \prod _ { i \in X } c _ { i } ^ { \alpha - 1 }\tag{4}
$$

它把概率质量放在那些具有 $|X|$ 个非零元素的向量 $\mathbf{c}$ 上。我们令 $\alpha$ 相对较大，以确保所有 $o \in X$ 的结果都有较大概率实际生成数据，也就是让概率质量主要落在满足 $c_i \approx c_j$ 的向量上。于是，在已经观察到前 $D$ 次结果之后，第 $D+1$ 次动作得到某个结果的预测概率，由于 Dirichlet 先验与多项分布共轭，因此可以写成标准形式：

$$
\begin{array} { r l r } { p ( n _ { D + 1 } = j | { \bf n } , X , \alpha ) } & { = } & { \left\{ \begin{array} { l l } { \frac { \alpha + n _ { i } } { | X | \alpha + N } } & { \mathrm { i f ~ } j \in X } \\ { 0 } & { \mathrm { e l s e } } \end{array} \right. } \end{array}\tag{6}
$$

需要注意的是，这只对我们条件化的集合 $X$ 内部的结果成立。给定式 (1) 中关于集合的先验，我们就能通过对集合大小求平均，得到任意结果的观测概率。要注意的是，那些不包含已观测结果集合 $Y$ 的集合，似然为零，因此不会对预测分布有贡献：

$$
\begin{array} { l l l } { p ( n _ { D + 1 } = j | { \bf n } , \alpha ) } & { = } & { \displaystyle \sum _ { X \geq \{ Y , j \} } p ( n _ { D + 1 } | { \bf n } , X ) p ( X | { \bf n } , \alpha ) } \end{array}\tag{7}
$$

$$
\begin{array} { l c l } { { p ( { \bf n } | X , \alpha ) } } & { { = } } & { { \displaystyle \int d { \bf c } p ( { \bf n } | { \bf c } ) p ( { \bf c } | X , \alpha ) } } \end{array}
$$

$$
{ \frac { N ! } { \prod _ { i \in X } n _ { i } ! } } { \frac { \Gamma ( | X | \alpha ) } { \Gamma ( | X | \alpha + N ) } } \prod _ { i \in X } { \frac { \Gamma ( \alpha + n _ { i } ) } { \Gamma ( \alpha ) } }\tag{8}
$$

$$
\begin{array} { r l r } { p ( X | { \bf n } , \alpha ) } & { = } & { \frac { p ( { \bf n } | X , \alpha ) p ( X ) } { \sum _ { X } p ( { \bf n } | X , \alpha ) p ( X ) } = \frac { B ( X ) } { \sum _ { X \geq Y } B ( X ) } } \end{array}\tag{9}
$$

$$
\begin{array} { r c l } { { { \cal B } ( X ) } } & { { = } } & { { \displaystyle \frac { \Gamma ( | X | \alpha ) } { \Gamma ( | X | \alpha + N ) } \prod _ { i \in X } \frac { \Gamma ( \alpha + n _ { i } ) } { \Gamma ( \alpha ) } \binom { L } { | X | } ^ { - 1 } p ( | X | ) } } \end{array}
$$

$$
\begin{array} { c c l } { \displaystyle \Rightarrow p ( n _ { D + 1 } = j | { \bf n } , \alpha ) } & { = } & { \frac { \sum _ { X \supseteq \{ Y , j \} } \frac { \alpha + n _ { j } } { | X | \alpha + N } B ( X ) } { \sum _ { X \supseteq Y } B ( X ) } } \end{array}\tag{10}
$$

式 (8) 是标准 Dirichlet 积分，式 (9) 是贝叶斯定理，而式 (10) 则给出了在已有观测基础上的预测分布。由于本文处理的问题中 $L$ 通常较小，例如大约为 6，所以这些求和可以显式计算；对于更大的问题，则可以从那些具有非零似然的集合 $X$ 中采样，以近似这些求和。

为了确认这一参数化确实会以可辨认的方式影响环境，我们基于一组来自独立动作的观测，使用 EM 对 $\zeta$ 进行推断。图 1 给出了结果。对于较大的 $\zeta$，即使样本很少，也能较准确地推断；但在较低 $\zeta$ 下，推断会更噪声。样本数较低时，似然似乎包含两个模态：一个在低 $\zeta$，一个在高 $\zeta$，用来解释那些少量出现了两个或更多结果的动作。然而，随着样本增加，第二个模态会迅速消失；若再加入哪怕一个弱先验，也能把它消除。

**补图 1.** 从 $L=20$ 个独立动作的观测中推断 $\zeta$，每个动作有 $L=10$ 个可能结果，并在 $\alpha=20$ 条件下对 $\mathbf{c}$ 求平均。

## 2. 具有独立结果的多动作情形

第二种控制概念不仅包含上面的结果熵，而且还进一步衡量：环境中的所有结果在多大程度上是可控可达的。也就是说，我们这里把控制定义为一种既考虑“不同动作是否可靠地产生结果”，也考虑“它们是否产生不同结果”，以及“这些结果是否覆盖了环境中可能出现结果范围”的量。

### 单个动作的结果分布

为了数学上的方便，我们只处理一类简化后的结果分布。我们把某个动作的条件分布简单参数化为“均匀分布与 Kronecker delta 的混合”。也就是说，对某个动作 $a$ 而言，若 $\mathbf{n}$ 表示观测计数，$n_i$ 是结果 $i$ 在选择动作 $a$ 后被观察到的次数，则有：

$$
P ( { \bf n } | c , { \bf m } ) = \frac { ( \sum _ { i } n _ { i } ) ! } { \prod _ { i } n _ { i } ! } c ^ { { \bf n } ^ { \mathrm { T } } { \bf m } } \bar { c } ^ { { \bf n } ^ { \mathrm { T } } ( 1 - { \bf m } ) } \bar { c } = \left( \frac { 1 - c } { L - 1 } \right)\tag{11}
$$

其中，$\mathbf{m}=[0\ldots010\ldots0]^T$ 用来指定某个结果是该动作的“可控可达”结果。标量变量 $c$（不要与上一小节中的结果概率向量 $\mathbf{c}$ 混淆）决定了混合成分的比例，我们称它调节了这个结果在多大程度上是“可控可达”的。没有被 $\mathbf{m}$ 指定的其他结果具有相等概率。这里 $L$ 是潜在结果数，为简化起见我们还假定可用动作数也同样为 $L$。

当 $c \to 1$ 时，只会观察到一个结果，也就是那个满足 $m_i=1$ 的结果；而当 $c$ 接近 $1/L$ 时，任何结果都可能出现。此时，这个动作的结果熵

$$
\mathcal { H } = - \sum _ { i } p _ { i } \log p _ { i } = - c \log ( c ) - ( 1 - c ) \log \frac { 1 - c } { L - 1 }\tag{12}
$$

对于 $L>2$ 是 $c$ 的严格单调递减函数。因此，$c$ 捕捉了原始的“结果熵式控制”概念。

### 多动作

对于一组独立动作，我们可以把观测的似然写成：

$$
P ( \mathbf { N } | \boldsymbol { c } , \mathbf { M } ) \propto \prod _ { a } c ^ { ( \mathbf { n } ^ { a } ) ^ { \mathrm { T } } \mathbf { m } ^ { a } } \bar { c } ^ { ( \mathbf { n } ^ { a } ) ^ { \mathrm { T } } ( 1 - \mathbf { m } ^ { a } ) } \propto \prod _ { i j } C _ { i j } ^ { N _ { i j } }\tag{13}
$$

其中，矩阵 $M$ 的第 $a$ 列向量 $\mathbf{m}^a$ 被分配给动作 $a$。矩阵 $M$ 的含义非常关键：每一列代表一个动作，每一行代表一个结果；某列中某个位置上的 1 表示该结果是该动作的主要“可控可达”结果。一个目标导向的行动者会选择该动作，以最大化获得该结果的机会。变量 $c$ 则决定：主体实际观察到这个指定结果，而不是其他结果的概率有多大。

第二种控制概念此时就体现在矩阵 $M$ 的各列之间的关系上，也就是不同动作的可控可达结果之间的关系。补充材料原文中给出了若干矩阵 $M$ 及其对应的矩阵 $C$ 作为例子，它们具有非常不同的含义。对于较大的 $c$，这些矩阵代表了一个可能的控制变量可以如何变化的若干维度：

- $\mathbf{M}_0$：结果 2 是可达的，但它也是唯一可达的结果。当 $c \to 1$ 时，所有动作都确定性地导向结果 2。
- $\mathbf{M}_1$：每个结果都有一个对应动作。当 $c \to 1$ 时，所有动作都能确定性地实现各自的结果，因此所有结果都是可控可达的。
- $\mathbf{M}_2$：只有一部分结果（例中为 3/4）有可达动作。
- $\mathbf{M}_3$：即便在 $c$ 很高时，动作也可能导向不止一个唯一结果，因此仍然不能视为完全控制。本文之后不再讨论这种设定。

当 $c$ 接近 $1/L$ 时，这些矩阵产生的观测都趋向于同样平坦、不可控的结果。稍后我们还会进一步考虑控制是否是“关于某些特别有强化价值的结果”的。

一旦考虑多个动作，就必须像式 (14) 那样把动作之间的关系纳入考量。我们回到式 (13) 的简单情形，并把矩阵 $M$ 约束为：每一列和每一行都恰好有一个 1。若动作数和结果数都为 $L$，这样的矩阵共有 $L!$ 个。对于较小的 $L$，相关积分可以显式求值。我们把观测似然写成式 (13) 的形式，并加入如下先验：

$$
\begin{array} { r c l } { p ( { \bf M } ) } & { = } & { \displaystyle \frac { 1 } { L ! } \left( \prod _ { j } \delta ( 1 - \sum _ { i } M _ { i j } ) \right) \left( \prod _ { i } \delta ( 1 - \sum _ { j } M _ { i j } ) \right) } \end{array}
$$

它保证每一行和每一列都只含一个 1。给定一组观测 $\mathbf{N}$，就可以写出关于 $M$ 的后验分布和对动作 $a$ 的预测分布：

$$
\begin{array} { r c l } { p ( \mathbf { M } | \mathbf { N } , c ) } & { = } & { \frac { p ( \mathbf { N } | c , \mathbf { M } ) p ( \mathbf { M } ) } { p ( \mathbf { N } | c ) } } \end{array}\tag{15}
$$

$$
\begin{array} { r c l } { p ( n _ { D + 1 } = j | { \bf N } , c , a ) } & { \propto } & { \displaystyle \sum _ { { \bf M } } c ^ { ( { \bf m } ^ { a } ) ^ { \mathrm { T } } { \bf d } ^ { j } } \bar { c } ^ { ( 1 - { \bf m } ^ { a } ) ^ { \mathrm { T } } { \bf d } ^ { j } } p ( { \bf M } | { \bf N } , c ) } \end{array}\tag{16}
$$

其中 $d_i^j=\delta_{ij}$。图 2A 展示了在三种不同 $c$ 值下，后验分布 $p(\mathbf{M}|\mathbf{N},c)$ 的均值。由于 $c$ 在动作之间共享，而 $M$ 又假定所有结果都可达，因此这意味着：若 $c \to 1$，所有结果都能由唯一动作实现；若 $c$ 接近 $1/L$，则没有任何结果真正可控可达。图 2A-D 就展示了这种约束的效果。

为了放松这一假设，我们允许具有可控可达结果的动作数 $|M|$ 变化。也就是说，矩阵 $M$ 的每一行和每一列可以有一个 1，也可以没有，正如原文式 (14) 中的 $\mathbf{M}_2$ 所示。与上一小节类似，我们对可控可达结果数 $|M|=\sum_{ij}M_{ij}\leq L$ 放置先验 $p(|M|)$，再对其积分，得到关于矩阵的先验：

$$
\begin{array} { r c l } { { p ( { \bf M } ) } } & { { = } } & { { \displaystyle \sum _ { | M | = 1 } ^ { L } p ( | M | ) \left[ \left( \begin{array} { c } { { L } } \\ { { | M | } } \end{array} \right) \frac { L ! } { ( L - | M | ) ! } \right] ^ { - 1 } B ( { \bf M } ) \delta \left( \sum _ { i j } M _ { i j } - | M | \right) } } \\ { { { } } } & { { } } & { { } } \\ { { { \cal B } ( { \bf M } ) } } & { { = } } & { { \displaystyle \left( \prod _ { j } \left[ \delta \left( 1 - \sum _ { i } M _ { i j } \right) + \delta \left( \sum _ { i } M _ { i j } \right) \right] \right) \times } } \\ { { } } & { { } } & { { \displaystyle \left( \prod _ { i } \left[ \delta \left( 1 - \sum _ { j } M _ { i j } \right) + \delta \left( \sum _ { j } M _ { i j } \right) \right] \right) } } \end{array}\tag{17}
$$

这里 $B(\mathbf{M})$ 保证矩阵的每一行和每一列最多只有一个 1。式 (17) 中，我们让所有具有相同非零元素数的矩阵具有相同先验概率。对于一个 $L \times L$、且 $|M|=k$ 的矩阵，选择列的方式共有 ${\binom{L}{k}}$ 种，而给这些列填充值的方式共有 $L!/(L-k)!$ 种，因为顺序是有区别的。

为了做预测，我们需要基于数据求出“可控可达结果数量”的后验分布。同时，为了确认这确实是一个可逆的生成模型，我们也希望能够对其进行推断。对应后验为：

$$
\begin{array} { r l r } { p ( | { \cal M } | = k | { \bf n } , c ) } & { = } & { \frac { \sum _ { { \bf M } : | { \cal M } | = k } p ( { \bf N } | { \bf M } , c ) p ( { \bf M } | k ) } { \sum _ { | { \cal M } | } p ( | { \cal M } | ) \sum _ { { \bf M } : | { \cal M } | = k } p ( { \bf N } | { \bf M } , c ) p ( { \bf M } | k ) } \quad \quad } \end{array}\tag{18}
$$

因此，如果先验 $p(|M|)=\delta(|M|-L)$，我们就回到了之前“所有结果在 $c$ 足够大时都必须可达”的设定；若先验在较小的 $|M|$ 上也有质量，那么并非所有结果都会有专门动作与之对应。图 2E-H 说明了降低 $|M|$ 所带来的影响。

作为进一步检查，我们还会在观测条件下，用 EM 推断 $c$ 和 $|M|$ 的最大似然设定。图 3A 显示了从数据中推断 $c$ 的特性，结果非常准确。我们也会进一步考察基于 $|M|$ 的泛化特性，因此希望能够推断它。图 3B 和 3C 展示了这种推断的表现：在观测数较少时，低控制设定的证据自然较少，因此 $|M|$ 往往会被高估。

**补图 2. 可达结果比例的先验。** 左边几幅图展示了熵参数 $c$ 和先验假定的可达比例如何共同影响预测分布。图 B-D 显示的是：在图 A 的观测数据条件下、并且先验假定所有结果都可达时，不同 $c$ 值对应的关于矩阵 $M$ 的后验均值。$c$ 越高，后验均值越被满足约束条件的单一矩阵主导。虽然动作 6 尚未被观察到任何结果，但在较高控制水平下，由于“所有动作都导向不同结果”这一约束，它的结果仍会被高确定性地推断出来。右边几幅图则展示放松先验、减小 $|M|$ 的效果：此时不同动作之间不再能通过“排除法”互相约束，因此不确定性增大。尤其是对于未观测到结果的动作，其后验均值会变平。

## 2.1. 多动作情形下的探索、激励对比与平均奖励

为了说明这种设置与前面那个忽略动作间关系的设定之间的共同点和差异，我们再次回到售货机例子。设有 $|A|=5$ 个按钮、$L=5$ 个可能结果，并且我们允许按下 $D=4$ 次按钮。按下按钮 $o$ 倾向于产生结果 $o$。过去只按过一个按钮（按钮 1）共 4 次，而且它每次都产生结果 1，对应奖励 0。设各结果的奖励为 $R=[0, 0.2, 0.23, 0.27, 0.3]$，这样一来，未探索动作的期望值 $\frac{1}{L}\sum_o R_o = 0.196$ 略低于第二个动作对应的奖励。其余四个按钮 $a_i$ 会以最高概率产生对应结果 $o=i$，因此实际上按钮 5 才是最佳动作，只是我们并不知道这一点。

为了便于说明，我们强行规定探索要按顺序进行：如果决定尝试一个新按钮，就必须尝试序列中的下一个按钮，而不能直接跳到按钮 5。于是，探索深度，也就是探索在哪个动作处停止，就成为一个衡量“探索驱力”的指标。图 4A 展示了不同先验对探索深度的影响。这里的先验是硬约束，只允许与某个特定 $|M|$ 精确一致的矩阵。

- $|M|=0$：我们相信没有任何按钮会稳定导向某个结果。即便已经看到按钮 1 四次都产生结果 1，对所有动作（包括按钮 1）而言，预测分布仍然是平的。按钮 1 看起来和其他完全未知的按钮一样好。图 4A 显示，在这个先验下，四次选择都会落在按钮 1 上。
- $|M|=1$：我们相信只有一个按钮会可靠地导向某个结果。此时 $M$ 的最大似然估计只在按钮 1/结果 1 这一位置有一个非零元素，其余按钮都被认为会随机地产生任意结果。由于我们的奖励设置，按钮 2 此时比按钮 1 略有优势，因此第一次会选择按钮 2；之后仍会继续选择按钮 2，因为它的结果（结果 2，对应奖励 0.2）略高于那些未知动作的平均值。
- $|M|=2$：我们相信有两个按钮会可靠地导向两个不同结果。这样一来，第二个非零元素就有可能被分配给按钮 3/结果 3。按钮 3 的预测分布不再是平的，因此其期望结果会高于按钮 2。不过，如图 4A 所示，这种探索大多会在按钮 3 处停止。
- 随着矩阵 $M$ 被约束为在不同列中包含越来越多的非零元素，也就是我们相信越来越多的结果可以通过某个按钮实现，探索就会持续下去，直到所有按钮都被试过。

这些探索效应，来自图 2 所展示效应的一个连续化版本。图 4B 和 4C 还表明，和前一个设定类似，当先验在更大的 $|M|$ 上放置更多概率质量时，平均 Q 值会上升，而且动作之间的期望奖励差异也会增大。

**补图 4. 关于可控结果比例的先验对探索和期望奖励的影响。** (A) 探索：还剩 $D=4$ 次动作，环境有 5 个动作和 5 个结果。各柱分别表示在第一、第二、第三和第四次试验中平均选择了哪个动作。随着先验假定更多结果是可控可达的，探索会走得更远。当先验假定所有动作都是可控可达的（$|M|=5$）时，有 40% 的情况下最终会探索到动作 5；而在 $|M|=4$ 时，这一比例只有 5%。(B) 不同 $|M|$ 先验下的平均 Q 值。平均而言，先验假定更大比例的结果可控可达，会得到更高的期望奖励。(C) 各状态下 Q 值的方差。高控制先验会导致动作价值之间的差异更大，也就是更大的激励对比。

## 3. 对理想结果的控制

现在我们来定义第三种控制概念，它是“总可获得正强化中，有多少比例是可控可达的”这一函数。令 $\mathbf{R}$ 为所有动作对应结果的强化向量。为了确保该定义同时适用于惩罚和奖励，先定义 $\tilde{\mathbf{R}}=\mathbf{R}-\min_i R_i$，再把每个结果的正向强化比例定义为 $\mathbf{r}=\tilde{\mathbf{R}}/\sum_j \tilde{R}_j$。当然，也可以用最大强化来归一化。于是，一个矩阵 $M$ 允许主体控制其中 $\mathbf{r}^T \mathbf{M}$ 这一部分强化；给定数据后，可控可达强化的平均比例可写为：

$$
\chi _ { M } = \sum _ { i } r _ { i } \sum _ { j } \left[ \mathbb { E } [ \mathbf { M } | \mathbf { N } , c ] \right] _ { i j }\tag{19}
$$

如果期望是对“每列至多有一个 1”的矩阵集合取的，那么有 $0 \leq \chi_M \leq 1$。这一一定义与 $c$ 的关系有些奇特：$c$ 被用于构造后验 $p(\mathbf{M}|\mathbf{N},c)$，但之后又被忽略了。例如，当 $c$ 接近 $1/L$ 时，后验均值仍可能被一个满秩的 $M$ 主导，这会意味着“可控可达结果比例很高”，尽管每个动作其实都几乎不偏向任何特定结果。这个问题可以通过一个线性映射来修正：

$$
\chi \quad = \quad \frac { L c - 1 } { L - 1 } \chi _ { M }\tag{20}
$$

这样就把 $c$ 也完整纳入了 $\chi$ 之中。若给定奖励向量 $\mathbf{R}$，就可以把某个特定的 $\chi$ 值作为关于矩阵 $M$ 的一个额外线性约束。于是，不再使用式 (17)，而改写为：

$$
\begin{array} { r l r } { p ^ { * } ( c , \mathbf { M } | \chi ) } & { = } & { \exp \left( - \frac { ( \chi - \frac { L c - 1 } { L - 1 } \mathbf { r } ^ { \mathrm { T } } \mathbf { M } \mathbb { I } ) ^ { 2 } } { 2 \sigma ^ { 2 } } \right) } \end{array}\tag{21}
$$

其中，$\mathbb{I}$ 是一个全 1 列向量，上标 $*$ 表示这是未归一化量。参数 $\sigma$ 决定对 $\chi$ 施加约束的严格程度。我们将通过重要性采样对 $c$ 上的积分进行求值，并通常令 $\sigma$ 很小。

给定前 $D$ 次观测后，新事件 $n_{D+1}$ 的预测形式为：

$$
\begin{array} { r l r } { p ( n _ { D + 1 } | { \bf N } , \chi , { \bf r } ) } & { = } & { \displaystyle \sum _ { { \bf M } } \int d c p ( n _ { D + 1 } | { \bf M } , c ) \frac { p ( { \bf N } | { \bf M } , c ) p ( c , { \bf M } | \chi ) } { \sum _ { { \bf M } } \int d c p ( { \bf N } | { \bf M } , c ) p ( c , { \bf M } | \chi ) } } \end{array}\tag{22}
$$

图 5 表明，$\chi$ 同样会对观测产生可识别的影响。

在主文用于习得性无助的应用中，我们会在若干个点上计算后验概率，并构造一个后验分布的近似：

$$
p ( \chi | { \bf N } ) \approx \sum _ { i } w _ { i } \delta ( \chi - \chi _ { i } )\tag{23}
$$

$$
\begin{array} { r c l } { w _ { i } ^ { * } } & { = } & { \displaystyle \frac { p ( \chi _ { i } ) } { p ( \mathbf { N } ) } \sum _ { \mathbf { M } } \int d c p ( \mathbf { N } | \mathbf { M } , c ) p ( c , \mathbf { M } | \chi _ { i } ) } \end{array}\tag{24}
$$

其中 $w_i = w_i^*/\sum_j w_j^*$。也就是说，后验被表示成若干 delta 函数之和，而积分通过重要性采样计算。

**补图 5.** 使用 EM 从结果观测 $\mathbf{N}$ 中推断可控强化参数 $\chi$。在该参数的整个取值范围内，都能够比较准确地恢复它。
