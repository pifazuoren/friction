# Paper Collection for `digital_helplessness_proto`

这份目录按你现在的研究方向做了一个偏“可直接用于 prototype / RP”的文献池，重点放在三类：

1. LLM social simulation / agent-based simulation 方法论文献
2. helplessness / controllability 的理论基础
3. 老年人数字焦虑、technophobia、数字健康采纳、数字支持与数字摩擦

## 1. 方法论：LLM 与社会模拟

- `AgentSociety_2025_Large-Scale_Simulation_of_LLM-Driven_Generative_Agents.pdf`
- `LLM_Agents_for_Piloting_Social_Experiments.pdf`
- `generative_agents_interactive_simulacra_of_human_behavior_2023.pdf`
- `social_simulation_driven_by_llm_agents_survey_2024.pdf`
- `gensim_general_social_simulation_platform_with_llm_agents_2024.pdf`
- `using_llms_to_simulate_multiple_humans_2022.pdf`

这组主要用于回答：

- 为什么可以用 LLM agent 做社会行为模拟
- 为什么 prototype 阶段可以先做小型可控实验
- 你的实验在“simulation validity / pilot experiment”上的方法论位置是什么

## 2. 理论基础：helplessness / controllability

- `learned_helplessness_and_learned_controllability_review_2025.pdf`

这篇最适合拿来支撑你后面“无助感更新机制”的理论起点，尤其是：

- 负反馈累积
- controllability / uncontrollability
- 情绪与行为回避之间的关系
