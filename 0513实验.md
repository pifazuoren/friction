## 2. 总体结论

这组实验整体是 **通过 sanity test 的**。

最重要的结论有四个：

1. 外显行为方向合理：高摩擦低支持世界最差，低摩擦高支持世界最好。
2. Bayesian policy audit 完整：policy/control payload coverage 都是 1.0。
3. 没有发现 pre-outcome policy 偷看 outcome 后信息：`uses_post_outcome_information_true_count = 0`。
4. gated-lite 介入非常克制：总体介入率约 7.47%，平均 TVD 只有 0.001487。

作为 AAAI/AISI 审稿视角，这组结果可以支持：

> Bayesian gated-lite2 已经是一个可审计、低 fallback、低扰动的行动学习层。

但它还不能单独支持：

> Bayesian 学习机制已经显著主导行为变化。

原因是这组实验中 `pi_final` 和 `pi_ref` 很接近，Bayesian gated shift 平均幅度很小。它更像一个安全、温和、可解释的调节器，而不是强行为学习引擎。

## 7. 为什么 intervention_applied_rate 只有 7.47%

`intervention_applied_rate = 0.074688` 看起来不高，但这是设计结果，不是 bug。

Bayesian 每条记录都运行了：

```text
policy_payload_coverage = 1.0
```

但只有满足 gate 条件时，才会真正改变最终策略：

```text
confidence(action) >= 0.50
posterior_entropy(action) <= 0.85
```

本次真正限制 gate 的是 confidence，不是 entropy。

代码中的 confidence 形式是：

```text
confidence = update_count / (update_count + confidence_k)
confidence_k = 4
```

也就是说，某个 `task_family + action` 至少需要累积几次真实经验，confidence 才能接近或超过 0.50。

这次实验里经验被分散到：

```text
6 task families
3 actions
10 agents
10 days
```

而且每次只更新真实发生的 action。比如 agent 选择了 `seek_help_then_attempt`，就只更新 help 的 posterior，不会反向更新 attempt_self 或 avoid。

因此，前几天多数 `task_family + action` 的证据还不够，gate 不会打开。

按 action 计算：

| action | confidence >= 0.50 的比例 |
|---|---:|
| attempt_self | 0.005166 |
| seek_help_then_attempt | 0.032932 |
| avoid | 0.037667 |
| 任一 action 达到 gate | 0.074688 |

这正好对应总体 `intervention_applied_rate = 0.074688`。

## 11. 作为 AAAI/AISI 审稿人的判断

如果只看这组实验，我会给出的评价是：

### 优点

第一，机制是可审计的。每条 attempt 都记录了 Bayesian policy payload，而且能看到 `pi_prior / pi_llm / pi_semantic / pi_ref / pi_bayes / pi_final` 的分工。

第二，工程 fallback 很低。semantic fallback 约 0.32%，rule fallback 和 safety fallback 都是 0。这说明当前不是 rule-heavy fallback。

第三，没有明显信息泄漏。pre-outcome Q/policy 没有使用 outcome 后的 support_mode、attribution、actual outcome 等字段。

第四，world-level 行为方向合理。高摩擦低支持最差，低摩擦高支持最好，高摩擦高支持处于中间。

第五，Bayesian posterior 的方向也合理。高摩擦低支持下 `pi_bayes_avoid` 和 `shadow_top_avoid_share` 更高，低摩擦高支持下 shadow avoid 很低。

### 主要不足

第一，Bayesian 介入仍然偏弱。平均 TVD 只有 0.001487，说明最终策略几乎没有被强烈改变。

第二，当前更能证明“安全接入”，还不能强证明“Bayesian learning 对行为有显著贡献”。

第三，paired seeds 只有 3，统计检验不足。当前结果更像 pilot evidence，而不是最终主实验。

第四，当前仍然是 single steady stage。learned helplessness 更适合 staged design，例如 baseline acquisition、uncontrollable friction exposure、support recovery、transfer/scope spillover。

第五，support 仍然主要是 world-level condition，还没有进入真正的 helper agent / support ecology 互动机制。

## 12. 这组实验最适合怎么写进论文

推荐写法：

> We first conduct a gated-lite stress test with semantic_v2 reference, lambda_llm = 0.50, and max_delta = 0.10. The audit shows complete policy/control payload coverage, no post-outcome leakage, near-zero rule and safety fallbacks, and bounded policy perturbation. Behavioral outcomes remain directionally consistent across digital friction and support conditions.

中文理解：

> 我们先做了一组较强参数下的压力测试。结果说明机制接入是稳定、可审计、没有明显泄漏的；同时，外显行为仍然符合高摩擦/低支持更差、低摩擦/高支持更好的理论预期。

不建议写成：

> Bayesian RL 已经学到了最优策略。

更准确的表述是：

> Bayesian gated-lite provides a bounded, auditable action-outcome modulation layer for LLM social simulation.

## 13. 后续建议

### 13.1 短期优先

继续跑 sensitivity：

```text
lambda_llm = 0.10 / 0.25 / 0.50
max_delta = 0.05 / 0.10
gate_threshold = 0.50
confidence_k = 4
```

重点比较：

- fallback rate；
- mean/max TVD；
- intervention_applied_rate；
- world-level 外显行为；
- `pi_bayes(t)` 是否预测 `action(t+1)`；
- `pi_bayes(t)` 是否预测 `avoidance_delta(t+1)` 或 `helplessness_delta(t+1)`。

### 13.2 中期优先

扩大 paired seeds：

```text
n_pairs >= 10
最好 n_pairs = 20
```

否则 p 值大概率仍然卡在 0.25 附近。

### 13.3 机制优先

下一步不建议盲目增大 agent 数量，而是优先加 staged design：

```text
stage 1: manageable baseline
stage 2: repeated high friction / uncontrollable exposure
stage 3: support intervention / recovery
stage 4: transfer or scope spillover test
```

这样 learned helplessness 的 acquisition、generalization、recovery 会更清楚。

### 13.4 多 agent 改造方向

完成 gated-lite2 后，可以把 support 从 world scalar 改成 pre-outcome support_response：

```text
older adult chooses seek help
-> helper/environment generates support_response before outcome
-> outcome model uses support_response
-> Bayesian posterior updates actual action-outcome memory
```

这样 support_mode 不再是 outcome 后推断，而是求助行为之后、结果发生之前的真实互动过程。

最小 helper agent 可以先只做三类响应：

```text
enabling support: 教老人怎么做，保留自主性
substituting support: 直接替老人完成，短期成功但可能削弱自我效能
low-quality support: 敷衍、误导、模板回复或不耐烦
```

这会让实验更像 LLM social simulation，也更适合 AAAI/AISI 的机制叙事。

## 14. 最终判断

这次 2026-05-13 实验可以记录为：

```text
Bayesian gated-lite2 semantic_v2 stress test: passed.
```

更具体地说：

- 行为方向合理；
- audit 记录完整；
- 没有发现 post-outcome leakage；
- semantic fallback 很低；
- rule fallback 和 safety fallback 为 0；
- Bayesian intervention 随经验积累逐步打开；
- 策略扰动幅度很小，保留了 LLM simulation 主体。

但这还不是最终论文主结果。它最适合承担的角色是：

```text
机制接入验证 + 工程 sanity check + sensitivity evidence
```

后续要冲 AAAI/AISI，需要继续补：

- 更多 seeds；
- sensitivity / ablation；
- lagged prediction；
- staged learned helplessness design；
- support ecology 或 helper agent 的最小多 agent 改造。
